About This Project

Project Description

This Final Year Project applies sentiment analysis and data visualization techniques to analyze book reviews. By embedding Power BI dashboards into the website, users can explore trends, sentiments, and recommendations through interactive visuals.

Project Mission

To develop an interactive book review analytics system that integrates Power BI dashboards into a web platform, enabling users to gain meaningful insights from book review data effectively.

Project Vision

To support data-driven reading insights by providing an accessible, user-friendly, and visually engaging analytics platform for book reviews.
